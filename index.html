50 Projects in 50 Days: HTML, CSS, and JavaScript

50 Projects in 50 Days is a course that was developed by Brad Traversy and Florin Pop which is available on multiple platforms. I accessed it via O'Reilly's learning platform (which I have access to via the university I'm taking a technical writing certificate course through) but it can also be found on Udemy and Traversy Media.

This portfolio is a collection of the projects that I created. I've included a short write up about what I learned when doing them and what techniques were used in building them.
